Projects

Gamified Chatroom: PixelPalz

Pixel Palz is a online web communication service which allows users to make friends with strangers that have similar interests online. RPG elements will be incorporated into the website to motivate usability and in-chat games will help users to connect with each other faster. This project will be helpful for users that want simplicity when it comes to making friends by eliminating the need to find and join servers/groupchats.

VR FPS Shooter Game

Our project will focus on player versus zombie beat-em-up gameplay. The game scene will take place in a post-war era where dead soldiers have risen from the grounds as zombies. Given different weather elements, the player is placed within a zoned area to survive inside that territory from incoming zombie attacks. Zombies- dressed in soldier uniforms- will spawn outside the protected zone, moving in and aiming to kill the players. For successful zombie kills, there are opportunities for a supply drop including but not limited to ammunition, health supplies, and currency for the player to redeem different weapon choices. In each scene, the weather will create a combination of different disadvantages for the player, such as slowed movement or lowered visibility. The player will have to strategize according to the mechanics of the game and utilize their environment to successfully survive the duration of the game.

School Campus Management System

This project, Campus Management System, is a full-stack web application using the PERN (PostgreSQL, Express, React, Node.js) technology stack. The system will have a database able to handle CRUD operations Create, Read, Update, and Delete records using Sequelize and handle functions for request processing. Site users should be able to see a list of campuses, add new campuses, edit campuses, and remove campuses. Additionally, there is also information about each of the students who attend individual campuses with the same ability to view, add, edit, and remove student information. The website will be simple and intuitive to use for the user.
